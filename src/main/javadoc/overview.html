Short overview of the Informaticum's XJC Plug-Ins.

The Informaticum's XJC plug-ins allows you to enrich the code generated by the Java XJC plug-ins. Just put them into the code generating process and activate them by choosing the according XJC option. Visit the project's GitHub page to learn about the details.

Informaticum's XJC Plug-Ins for Developer

If you plan to develop your own XJC plug-ins you might want to reuse the code of this project. Probably, you should start extending the {@link de.informaticum.xjc.api.BasePlugin} or any of its sub-classes.

If there is any bug or missing code, feel free to improve this project. Again, see the project's GitHub page to read about the details.
